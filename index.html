Middleware Software Information

Name Description Type Advantages Examples
Remote Procedure Call (RPC) Enables executing functions on remote servers as if they were local. Communication Middleware Simplifies remote communication, abstracts network complexities. ONC RPC, Java RMI
CORBA (Common Object Request Broker Architecture) Standardizes communication between distributed objects across languages. Middleware Platform Supports interoperability, language independence. OMNIORB, JacORB
Message-Oriented Middleware (MOM) Facilitates asynchronous communication using messages/events. Messaging Middleware Loose coupling, reliability, scalability. Apache Kafka, RabbitMQ
